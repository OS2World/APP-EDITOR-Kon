Key features

  • Fully programmable using a C-like programming language (K)
  • Enter K-expressions immediately from within Kon, no need for separate compiling steps
  • Supports both Ascii and Hex view
  • Configurable syntax highlighting. (Can define syntax highlighting using regular expressions)
  • Configurable key bindings
  • Possible to modify main menu, or create your own using K.
  • Custom context-sensitive pop-up menus, all configurable using K
  • Search or Search-and-replace in current file, opened files or on disk (also in subdirectories)
  • Possible to add column separators
  • Runs on both Windows, OS/2 and eComStation
  • Recorded macros can be saved as a K-function to file, or be copied to clipboard
  • Can run K-code from clipboard
  • Autosaves files
